iOS安全（五）之iOS7 新增安全功能

2014-01-16
iOS

  • 引言

    大家都知道，苹果在2013年发布了最新一代iPhone 5S以及iOS 7系统。iOS 7有着完全重新设计的用户界面，我个人是很喜欢这种设计，在beta版本发布的时间就第一时间升级了。所以本文中，我们将介绍iOS 7中引入的新的安全功能。

  • 激活锁

    在以前，如果用户丢失了他们的iPhone，他们可用“查找我的iPhone”，用GPS来跟踪他们的手机。不过以前这个流程很有问题，任何人拿到了iPhone以后都可以将iPhone恢复到出厂设置，或者用iTunes将设备恢复，所有的内容和设置都会被清空。在iOS 7，有了新的激活锁这个功能，要执行恢复出厂设置，就必须知道用户Apple ID对应的密码。就算用iTunes将iPhone的固件重刷恢复，要激活也需要输入用户的Apple ID和密码。

要验证这个功能，请到“设置”->“通用”，滚动到底部，你可以看到一个“重置”按钮：

点击这个按钮，然后点击擦除所有内容和设置，会弹出一个框，要求你输入Apple ID对应的密码。这是个很实用的功能，可以防止别人（小偷）重置你iPhone的内容和设置。你如果想去“设置”->“iCloud”->“关闭查找我的iPhone”，也会得到同样的弹出框：

  • Airdrop

    Airdrop用于在iOS设备之间分享文件，只要将文件拖动到使用AirDrop功能的好友的头像上，就能进行一对一的文件传输，默认这项功能只对你的联系人可见。不过你也可以更改这个设置。AirDrop 使用 Wi-Fi 和蓝牙传输信息，您需要使用 iCloud 帐户来与联系人共享。好消息是，这些传输的信息都是经过加密的。要注意的是Airdrop只支持iOS 7以及iPhone 5或更新机型，所以笔者的iPhone 4S不能来演示这个功能，有设备的可以自己研究一下。

  • 保护第三方应用数据

    数据保护只有在提供硬件加密的设备上才可用，这些设备包括iPhone 3GS或更新机型、所有的iPad、第三代iPod Touch或更新机型。苹果的官方文档是这样描述的：

    数据保护通过用你的密码来保护硬件加密秘钥，增强了内建的硬件保护。对邮件地址和附件来说，提供了另一层的保护。从iOS 4开始，第3方应用就可以使用数据保护APIs来保护应用的数据。
    在iOS 6中，你可以到“设置”->“通用”->“密码”来选择一个密码，你可以看到底部有个文本，说数据保护已经启用。这意味着苹果在iOS6中默认对它自己的应用开启了数据保护这个功能。而现在在iOS 7中，所有的第三方应用都自动开启了数据保护，这意味着用户解锁设备之前，所有应用内部的数据都是安全的。 我们知道，默认的，苹果允许4位数字密码，这个很容易被暴力破解。因此，为了安全，推荐使用更安全的密码。

  • iCloud Keychain

    大部人可能都在使用或使用过Safari的自动填写功能。在iOS 7中，Safari可以记住更多信息，例如用户名，密码，信用卡号等等，并且把这些信息保存在Keychain中。然后就会在用户访问网页的时候自动填写。所有这些信息都是用256位的AES加密算法加密的。这个Keychain也会通过iCloud同步。为每一个你登录的站点都创建一个新密码会有点烦人，并且难以记忆。Safari内建了一个密码生成器，为你登录的每个不同站点创建不同的密码，你自己都不需要去记它。

  • Per App VPN

    有时候，用户需要通过一个安全通道取访问机密信息，因此他们使用VPN。在iOS 7中，苹果允许为不同应用使用不同的VPN。这非常有用，比如你想用公司的app通过公司提供的VPN来访问公司数据，而其他应用，你可能想要用其他VPN。这个功能对政府雇员也很有用，这样他们就可以通过手机应用来访问机密数据。

  • 总结

    本文总结了iOS 7在安全方面的新功能，这些新功能使用户在使用iOS设备时更放心。然后以上所说的这些功能都是在设备没有越狱的情况下，有些功能越狱之后可以很容易被破解掉。所以在平时使用设备时还是要注意。
